제안 배경
일반적으로 허위정보는 사람들의 관심을 끌기위해 사실여부와는 상관이 없으며 자극적인 경우가 많으며 빠르게 확산됩니다. 때문에 허위정보는 빠른 탐지와 대응이 중요합니다. 특히 해외에서 생성된 정보는 허위사실 탐지 및 대응이 더욱 어렵습니다. 이런 정보를 정확하게 판단하고 신속하게 대응하기 위해 NLP 모델 기반으로 허위 여부를 판단하고, 정보를 실시간으로 검색하는 솔루션이 필요합니다. 



아이디어 개요
제안할 아이디어는 텍스트의 문맥과 유사도를 평가하기 위해 KoBERT기반 벡터화를 사용하고, 외부 패특체킹 API와의 연계를 통해 허위정보 여부를 판별합니다. 



0. 허위정보라고 추정되는 텍스트를 입력합니다.

1. 유사단어 검색을 위한 벡터화: 유사단어에 대한 검색을 사용하기 위해 텍스트를 분석합니다. 분석 모델로는 트랜스토머 기반 언어 모델 중 한국어 처리에 특화된 KoBERT 모델을 이용해 텍스트를 벡터화합니다. 
벡터화한 값에 가중치를 두어 중요 단어를 뽑아 검색합니다. ( 검색문서는 뉴스기사로 한정하며 뉴스기사는 기자이름있고, 공신력있는 언론인지를 판단하여 가져옵니다. ) 

(코사인 유사도: 두 벡터 간의 유사도를 측정하는 방법으로 단어 벡터의 유사도를 계산하는 데 사용됩니다.)



2. 문서와의 연관성 찾기:  검색한 뉴스기사와의 유사도를 이용해 top-n개를 추출하고 이를 통해 문서와의 연관성을 파악합니다.


3. 텍스트가 허위정보일 가능성을 평가: 외부 팩트체킹 API 연계 (Google Fact Check Tools API)
API 실시간 검증: 검색기사 중에서도 허위정보로 의심되면 Google Fact Check Tools API를 통해 관련된 검증된 기사나 데이터를 검색합니다. 입력 텍스트가 포함된 문장이 이전에 검증된 내용과 일치하는지 확인합니다.


4. 유사도 기반 허위 판별 보완: Google API 검색 결과에서 추출한 정보와 비교해, 신뢰성 점수를 부여하고 판별 결과에 반영합니다. 


5. 신뢰도 점수와 유사도 결합을 통한 판별 기준 강화

신뢰도 점수 부여: 검색된 출처의 신뢰도에 따라 점수를 부여하여, 신뢰도가 높은 경우 판별 임계값을 낮추고, 낮은 경우 판별 기준을 강화합니다. 

신뢰도와 유사도 종합 평가: NLP 모델의 유사도와 외부 데이터의 신뢰도를 결합해 최종 판별을 수행하고, 시각적으로 허위 여부를 사용자에게 제공합니다.



6. 사용자 피드백 기반의 지속적 개선
판별 결과에 대한 사용자 피드백을 수집해 모델을 주기적으로 개선합니다. 피드백을 통해 다양한 표현 방식에 대한 탐지 정확도를 지속적으로 높여갑니다.



장점 및 기대효과
KoBERT와 GPT 모델 결합을 통해 텍스트의 문맥을 깊이 이해하고, 단순한 단어 일치가 아닌 의미 기반 탐지를 기대하고, Google Fact Check Tools API와 신뢰도 점수를 종합 평가해 허위정보를 판단합니다. 모든 정보를 사전 크롤링하지 않고 필요 시에만 API로 검색하므로, 빠르게 판단할 수 있습니다.
